## How to add report and slides

1. Save your Word report as `docs/report.docx`
2. Save your PowerPoint as `docs/presentation.pptx`
3. From the project root, run:

```bash
git add docs/report.docx docs/presentation.pptx
git commit -m "Add report and presentation"
git push origin main
```

Both files will then appear on GitHub under the `docs/` folder.
